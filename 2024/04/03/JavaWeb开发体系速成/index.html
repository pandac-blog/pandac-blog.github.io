Web

介绍

Web是什么

Web即全球广域网

也称万维网(www -> World Wide Web)

就是能通过浏览器访问的网站

工作流程

Web网站工作流程(前后端分离模式)↓↓↓

  • 组成内容

    前端服务器(前端程序)

    后端服务器(Java程序)

    数据库服务器

  • 工作流程

    1. 用户在浏览器通过域名向前端服务器(对应前端程序)发起请求,前端服务器响应返回前端程序代码(即展示样式),浏览器解析前端代码展示效果
    2. 前端服务器同时也会返回一个地址,表示获取数据的地址.则浏览器向后端服务器发起请求,后端服务器继续向数据库服务器发起请求
    3. 数据库服务器响应,返回数据至后端服务器,然后后端服务器响应将数据返回给浏览器(即展示数据)

  • Web网站开发模式

    前后端分离 -> 前端工程师开发前端程序,后端工程师开发后端Java程序

    混合开发(前后端不分离) -> 浏览器发起请求后,将前端程序与数据一并返回给浏览器

浏览器

前端代码经过浏览器的解析和渲染转化成用户看到的网页页面

浏览器内核

  • 即浏览器中对代码进行解析渲染的部分
  • 不同浏览器因为内核不同,解析效果会有差异

需要制定统一标准↓↓↓

Web标准

也称为网页标准

  • 大部分由W3C(万维网联盟)制定

组成部分

  • HTML : 负责网页的结构(页面的元素和内容)
  • CSS : 负责网页的表现(页面元素的外观样式)
  • JavaScript : 负责网页的行为(交互效果)

HTML

控制网页的结构

HyperText Markup Language,超文本标记语言

详情请看HTML精简笔记

基础标签

图片标签img

<img>

属性

  • src -> 图片的url
  • width/height -> 宽高

路径

绝对路径

  • 绝对磁盘路径 C:\xx\yy\zz.txt
  • 绝对网络路径 https://pandac.top

相对路径

  • /打开下级文件夹
  • ./打开当前文件夹
  • ../打开上级文件夹

标题标签h1-h6

<h1>-<h6>

水平分隔线标签hr

<hr>

CSS

控制网页的表现

Cascading Style Sheet,层叠样式表

详情请看CSS精简笔记

样式引入方式

行内样式 -> 将CSS代码写在标签的style属性中

内联样式 -> 将CSS代码写在title标签下的style标签中

外联样式 -> 将CSS代码写在.css文件中,通过link标签引入html文件

JavaScript

控制网页的行为

详情请看JS基础笔记

介绍

JS是一门跨平台,面向对象的脚本语言(不用进行编译,直接由浏览器解析执行)

ECMAScript是最新的JavaScript版本

引入方式

分类(三种)

  • 内部脚本

    将JS代码写在body标签最下面的script标签中

  • 外部脚本

    将JS代码写在.js文件中,通过src属性引入html文件中

  • 行内脚本

    将JS代码写在标签的属性中(不建议)

基础语法

书写语法

规则

  • 区分大小写

  • 结尾分号可有可无

  • 注释

    单行// ; 多行/* */

输出语句

window.alert() 顶部弹出警示框

document.write() 在html页面输出内容

console.log() 控制台输出内容

变量

JS中使用var关键字声明变量

var num = 10

  • var声明的变量可以重复声明
  • var声明的变量作用域是全局变量

而变量可以存储不同类型的值(因为JS是弱类型语言)

var num = '张三'

命名规则

  • 由字母,数字,下划线_,美元符号$组成
  • 不可数字开头
  • 小驼峰命名规范

注意事项

  • ECMAScript6新增了let关键字和const关键字来声明/定义变量

    let声明变量

    • let声明的变量的作用域是局部变量,仅在当前代码块有效
    • let声明的变量不允许再次声明

    const声明变量

    • 将变量声明为只读(不可更改)的常量

数据类型

原始类型(基本数据类型)

  • number数字类型 (整数,小数,NaN)
  • string字符串类型
  • boolean布尔类型 (true,false)
  • null空类型 (对象为空)
  • undefined未定义类型 (数据未初始化)

引用数据类型

使用typeof关键字返回指定数据的数据类型

运算符

算术运算符

​ + - * / % ++ --

赋值运算符

​ = += -= *= /= %=

比较运算符(关系运算符)

​ > < >= <= != == !== ===

​ ==会进行类型转换

​ ===全等(数据和类型都一致)

​ !== 不全等(数据或类型不一致)

逻辑运算符

​ && || !

三元运算符

​ 条件表达式 ? true返回值 : false返回值;

流程控制语句

if … else if … else

switch

for

while

do…while

函数

介绍

  • 被用来执行特定功能的代码块

定义

=>JS通过function关键字定义函数

注意事项

  • 形式参数列表不需要写参数类型(因为Js是弱类型语言)
  • 返回值也不用定义类型,直接return返回即可

调用

  • 函数名(实际参数列表)
  • 调用时可以传递若干个参数,但是只计算前n个实际有效参数

函数定义的第二种方式

对象

分类

  • 基础对象

    常用Array数组对象,String字符串对象,JSON对象

  • BOM对象

    浏览器对象模型

  • DOM对象

    文档对象模型

Array数组

定义

  • var 变量名 = new Array(元素列表);

    例如var arr = new Array(1,2,3);

  • var 变量名 = [元素列表];

    例如var arrStr = ['zhangsan','lisi','wangwu'];

访问

  • arr[索引/下标从0开始] = 值;

特点(和Java中集合很相似)

  • 长度可变
  • 类型可变

常用属性

  • length 设置/返回数组长度

    常与for循环结合使用,但是for循环遍历数组中所有元素,forEach只遍历有值的元素

常用方法

  • forEach() 遍历数组中所有有值的元素,并调入传入一次的参数

    简化写法ES6箭头函数

    forEach((e)) => {xx});

  • push() 将新元素添加到数组的末尾

  • splice() 从数组中删除元素

    • 第一个参数start_index(删除元素的起始索引)
    • 第二个参数number(删除个数),不写默认从index开始全删

String字符串

创建String对象

  • var 变量名 = new String('...');
  • var 变量名 = '...';

常用属性

  • length 字符串长度

常用方法

  • charAt() 返回字符串在指定索引的字符
  • indexOf() 检索字符串(即返回子字符串的首字符索引)
  • trim() 去除字符串首尾的空格
  • substring() 截取子字符串,截取并返回[起始索引,终止索引)这段子字符串
    • 第一个参数是起始索引
    • 第二个参数是终止索引

JS自定义对象

定义格式

调用格式

​ 对象名.属性名;

​ 对象名.方法名();

JSON

介绍

  • JavaScript Object Notation,JavaScript对象标记法

    JSON是通过JavaScript对象标记法书写的文本

JSON数据格式

​ {"x": "y"} 双引号包括键值对中的键,然后用:对应键值

作用

  • 由于语法简单,层次结构鲜明

    ==多用于作为数据载体,在网络中进行数据传输==

定义

  • var 变量名 = ‘{“key1”: value1, “key2”: value2}’;

    单引号’’包括大括号{},双引号””包裹键key

  • value值的数据类型

    数字,字符串(用双引号包裹),逻辑值(true,false),数组(用方括号[]包裹),对象(用大括号{}包裹),null空

常用方法

  • JSON字符串转为JS对象

    var jsObject = JSON.parse(userStr);

  • JS对象转为JSON字符串

    var jsonStr = JSON.stringify(jsObject);

BOM

介绍

  • Browser Object Model,浏览器对象模型

    运行JavaScript和浏览器对话,JS将浏览器的各个组成部分封装为对象

组成

  • Window -> 浏览器窗口对象
  • Navigator -> 浏览器对象
  • Screen -> 屏幕对象
  • History -> 历史记录对象
  • Location -> 地址栏对象

Window

介绍 -> 浏览器窗口对象

获取 -> 直接使用window关键字(其实可省略),然后调用方法名即可

属性

  • history 对History对象的只读引用
  • location 用于窗口或框架的Location对象
  • navigator 对Navigator对象的只读引用

方法

  • alert() 弹出警示框

  • confirm() 显示一段消息以及确认按钮和取消按钮的对话框

    例如var flag = confirm("确实删除该记录?")

    点击”确定”返回true, 点击”取消”返回false

  • setInterval() 按照指定的周期(毫秒单位)来调用函数或计算表达式

    第一个参数是要执行的函数,第二个参数就是一毫秒为单位的时间值 周期性执行函数

  • setTimeout() 在指定毫秒数后调用函数或计算表达式

    延迟x毫秒后执行函数,仅执行一次

Location

介绍 -> 地址栏对象

获取 -> 直接使用window关键字(其实可省略),然后调用方法名即可

属性

  • href 设置/返回浏览器地址栏的URL

    返回URL alert(location.href)

    设置URL然后跳转 location.href = 'https://pandac.top'

DOM

介绍

  • Document Object Model,文档对象模型

    将标记语言的各个组成部分封装为对应的对象

分类

  • Document -> 整个文档对象
  • Element -> 元素对象
  • Attribute -> 属性对象
  • Text -> 文本对象
  • Comment -> 注释对象

作用

  • JavaScript通过DOM,就可以对HTML文档进行操作

    改变HTML元素内容

    改变HTML元素样式

    对HTML DOM事件作出反应

    添加和删除HTML元素

组成部分

DOM定义了访问HTML和XML文档的标准,分为3个不同部分

  • Core DOM - 所有文档类型的标准模型

    Document -> 整个文档对象

    Element -> 元素对象

    Attribute -> 属性对象

    Text -> 文本对象

    Comment -> 注释对象

  • XML DOM - XML文档的标准模型

  • HTML DOM - HTML文档的标准模型

    Image: <img>

    Button: <input type='button'>

Element

通过Document对象获取Element元素对象

​ HTML文档内容如下

  • 根据id属性值获取,返回单个Element对象

    var h1 = document.getElementById('h1');

  • 根据标签名获取,返回Element对象数组

    var divs = document.getElementsByTagName('div');

  • 根据name属性值获取,返回Element对象数组

    var ins= document.getElementsByName('hobby');

  • 根据class属性值获取,返回Element对象数组

    var cls= document.getElementsByClassName('cls');

得到Element对象后,查看参考手册调用属性和方法即可

  • 比如将返回得到的cls对象数组中的div标签文本内容”PandaC”修改为”PandaC886”

    调用innerHTML方法

事件

介绍

  • HTML事件就是发生在HTML元素上的事情

    比如按钮被点击, 鼠标移动到元素上, 点击键盘按键

  • 事件监听

    JS可以在事件被侦测到时,执行对应代码

事件绑定

方式一

  • 通过HTML标签中的事件属性进行绑定

方式二

  • 通过DOM元素的事件属性进行绑定

常见事件

onclick 鼠标单击事件

onblur 元素失去焦点

​ 比如输入框光标从选中到离开,就算是失去焦点

onfocus 元素获得焦点

​ 比如输入框光标选中,就算是获得焦点

onload 某个页面或图像被完全加载完毕

onsubmit 表单提交事件

onkeydown 键盘按键事件

​ 常结合表单使用

onmouseover 鼠标移动到某元素上

onmouseout 鼠标从某元素上移开
