JS基础

1’JS介绍

JS是什么

JavaScript

  • 是一种脚本语言,即不用编译↓↓↓

  • 是一种运行在客户端(浏览器)的编程语言,能实现人机交互

  • HTML/css是标记语言,不是编程语言

作用

  • 网页特效(监听用户的一些行为让网页做出对应反应)
  • 表单验证(针对表单数据的合法性进行判断)
  • 数据交互(获取后台数据,渲染到前端)
  • 服务端编程(node.js)

组成

  • ECMAScript -> JS语言基础
    • 规定了JS的基础语法
  • WebApi
    • DOM -> 页面文档对象模型
      • 操作HTML文档,比如对页面元素进行移动
    • BOM -> 浏览器对象模型
      • 操作浏览器,比如页面弹窗

书写位置

CSS的引入方式有三种

  • 行内样式表
  • 内部样式表
  • 外部样式表

同理,JS的书写位置(引入方式)也有三种

  • 行内javascript
  • 内部javascript
  • 外部javascript

内部javascript

  • 直接写在HTML文件中
  • 在</body>标签上方,用<script>标签包裹住

例子(弹出警示框)

注意事项

  • js代码写在底部是因为要先获取html中的标签,再用js进行交互

外部javascript

  • 代码写在.js结尾的文件里
  • 通过script标签引入HTML页面

注意事项

  • 外部js引入时<script>和</script>标签中间不要写内容

行内/内联javascript

  • js代码写在标签内部

此方式了解即可,之后vue框架才会使用这种模式

注释

注释

  • 单行注释
    • //
    • 快捷键 -> ctrl+/
  • 块注释(多行注释)
    • /* */
    • 快捷键 -> shift+alt+a

结束符

作用

  • 使用英文;代表结束
  • 实际可写不写(浏览器会自动推断语句结束位置)
    • 要么都加要么都不加

输入输出语句

输出语法

  • document输出

    • 向body输出内容
    • 内容可以是标签元素

    document.write('内容')

  • alert输出

    • 向页面输出对话框

    alert('内容')

  • console.log输出

    • 向控制台输出内容

    console.log('控制台打印')

输入语法

  • 语法格式

    prompt('请输入内容')

  • 作用

    显示一个对话框,对话框中包含提示文字

  • 示例

JS执行顺序

  • 按HTML文档顺序执行JS代码
  • alert()和prompt()会跳过HTML渲染,先被执行

字面量

描述的数据值

比如10(数字字面量),’你好'(字符串字面量)等等

2’JS变量

存储数据的容器,而非数据本身

使用

声明/定义变量

  • 语法

    let 变量名

    • let -> 声明关键字
    • 变量名 -> 标识

变量赋值

  • 语法

    变量名 = 数据值/字面量

声明的同时直接赋值 -> 变量初始化

​ let age = 18

更新变量

  • 语法

    变量名 = 新数据值/字面量

声明多个变量

  • 语法(不建议)

    let 变量1 = 数值1, 变量2 = 数值2

本质

在内存空间中的一块空间/地址,其中存储了数据内容

这就是变量的本质

命名规则

规则

  • 不可使用关键字(被预定义功能的英文单词)
  • 只能用下划线_,字母,数字,$组成,且数字不可开头
  • 严格区分字母大小写

规范

  • 见名知意
  • 小驼峰命名法
    • 第一个单词首字母小写,之后所有单词首字母大写

let与var的区别

旧版本的JS中使用var声明变量

而现在使用let取代var来声明变量,以解决一些问题↓

  • var可以先使用再定义
  • 变量可以重复声明
  • 它就是个Bug,以后不要用var,要用let声明变量!!

3’JS数组

作用

  • 将一组数组存储在单个变量名下,方便使用
  • 可以存储任意数据(甚至元素不同)

语法格式

​ let 数组名 = [数据1, 数据2, ..., 数据n]

下标/索引

从0开始

取元素值

  • 语法

​ 数组名[下标]

长度属性

​ length 返回数组长度

4’JS常量

概念

  • 使用const 声明的变量就是常量

使用场景

  • 当某个变量的值不会再改变的时候,就用const声明为常量,而不用let声明为变量

语法格式

  • const PI = 3.14

注意事项

  • 常量不允许二次赋值(修改)
  • 产量声明不需要写let关键字

5’数据类型

数据类型分类

为数据分类,高效利用内存空间

JS是一门弱数据类型的语言 -> 只有赋了值才知道具体数据类型

  • 基本数据类型

    • number数字型

      • 不管是整数/小数还是正数/负数,都叫数字型
      • NaN(not a number) 代表一个计算错误
        • 其是”粘性”的,任何对NaN的操作得到的都是NaN

    • string字符串型

      • 使用规则

        • 使用单引号’',双引号””,反引号\'包裹的数据都是字符串(建议用单引号’’)
        • 单双引号可以互相嵌套(外单内双,外双内单),但是不能自己套自己
        • 必要时使用\转义输出’和”

      • 字符串拼接符号 -> +

        • 数字相加求和
        • 字符相加连接

      • 模板字符串

        • 使用场景 -> 拼接字符串和变量

        • 语法格式

          用反引号\'将${拼接内容}包裹起来

        • 例子

          document.write(/反引号我今年&{age}岁了/反引号)

    • boolean布尔型

      • 表示肯定/对错的数据
      • true为真,false为假

    • undefined未定义型

      • 只有一个值 undefined
      • 声明变量不赋值,则变量的默认值为undefined

    • null空类型

      • 代表一个”无/空”的特殊值
      • undefined代表没有赋值,而null代表赋值了但内容为空
      • null还可以代表一个对象,只是内容尚未赋值

  • 引用数据类型

    • object对象

检查数据类型

利用typeof关键字检测数据类型

作用

  • 返回被检测变量的数据类型

语法格式

  • 运算符形式(推荐)
    • typeof x
  • 函数形式
    • typeof(x)

数据类型转换

JS的接收数据默认字符串类型

比如prompt方法接收的数据,即使是数字,也会被处理为字符串

所以需要对数据类型进行转换

  • 隐式转换(自动转换)
  • 显式转换(强制转换)

隐式类型转换

  • 系统自动将数据类型进行转换

缺点

  • 规则不明显,依赖个人经验

规则

  • +号两边只要出现一个字符串类型,就会自动将另一边转为字符串类型

    +号妙用 => document.write(+'123') 结果是数字123

    ↑↑↑ 又称为一元加操作符

  • 除了+号算术运算符,- * /算术运算符参与运算都会自动把数据转为数字类型

显式类型转换

  • 程序员主动将数据类型进行转换

  • 转化为number数字类型

    1. Number

    • Number(目标数据) 将目标数据转为数字类型

    • 例子

      let age = +prompt('请输入年龄') 好好体会这行代码~

      原理:prompt传回来的数据默认字符串类型,利用+转为数字类型,然后赋给age变量,存储数字数据

    • 结果

      • number数字类型
      • NaN不存在这个数字
    1. parseInt

    • parseInt(目标数据) 将目标数据转为数字且只保留整数部分

    • 例子

      document.write(parseInt(3.14)) -> 结果3

      document.write(parseInt('3.14test')) -> 结果还是3

    1. parseFloat

    • parseFloat(目标数据) 将目标数据转为数字且保留小数部分

    • 例子

      document.write(parseFloat(3.14)) -> 结果3.14

      document.write(parseFloat('3.14test')) -> 结果3.14
