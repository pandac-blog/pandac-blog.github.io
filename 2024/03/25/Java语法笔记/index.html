第一章-Java入门

1’cmd

什么是cmd

  • Windows操作系统中,利用命令行的方式操作计算机的方法

cmd的常见操作

  • 打开方式
    • win+r打开
    • 到文件目录页上方输入cmd打开
  • 盘符名称+: -> 跳转到指定盘符
  • dir -> 查找目录下所有内容
  • cd 目录 -> 进入单级目录
  • cd .. -> 回退上一级目录
  • cd 目录1\目录2\... -> 进入多级目录
  • cd \ -> 回退到当前盘符根目录
  • cls -> 清屏
  • exit -> 关闭命令提示符窗口

利用cmd命令打开QQ

  1. 找到QQ程序位置
  2. 设置QQ的环境变量
    • 使得QQ能在任意文件夹下被打开
  3. 在cmd中打开QQ

环境变量

  • 记录某个程序的位置,存储到环境变量中
  • 就可以在任意文件夹下找到并且打开这个程序
  • 配置方法
    • 高级系统设置->环境变量
    • 系统变量path->编辑
    • 新建一个位置填入程序的路径

2’Java基础概念

Java是什么

  • 一种计算机语言

JDK的安装与配置

  • JDK目录
    • bin -> 存放各种工具命令
      • javac -> 编译工具,生成字节码文件
      • java -> 运行字节码文件
    • conf -> 存放了相关配置文件
    • include -> 存放平台特定头文件
    • jmods -> 存放各种模板
    • legal -> 存放各模板的授权文档
    • lib -> 存放了工具的一些补充jar包
  • 配置环境变量
    • 目的:在任意目录下都能打开指定程序
    • 安装JDK17时会自动配置Path环境变量(只有javac,java,javaw,jshell这四个工具)
    • 手动配置方法:
      • 先配置JAVA_HOME系统变量,存放jdk目录地址
      • 再配置Path下的
      • %JAVA_HOME%\bin环境变量,表示引用JAVA_HOME中的bin文件夹

Java三大类

  • JavaSE -> 标准版
  • JavaME ->移动版(已凉凉)
  • JavaEE ->企业版

Java跨平台原理

  • 高级语言编译运行方式
    • 编程:Java程序员写的.java代码,c程序员写的.c代码,python程序员写的.py代码
    • 编译(即”翻译”):机器只认识01的机器语言,需要吧.java,.c,.py的代码转化成机器语言
    • 运行:让机器执行机器指令
  • 高级语言的编译运行方式
    • 编译型:如c/c++语言,编写.c文件,然后编译为.obj文件,然后运行
    • 解释性:如python语言,.py文件按行翻译执行,不生成别的文件
    • 混合型(半编译+半解释):如java语言,先把.java文件整体编译成.class文件(javac),再按行把代码交给虚拟机运行(java),可以在不同平台的虚拟机中执行.java文件,所以具有良好跨平台性

特点:

3’字面量

告诉程序员:数据在Java程序中的书写格式

  • 比如整数就是28,小数就是14.13,字符就是’a'等等…

  • 在程序的执行过程中,其数据有可能发生改变的量

定义格式

数据类型 变量名 = 数据值;

  • 数据类型 -> 为空间中存储的数据加上类型的限制
  • 变量名 -> 为空间起的名字
  • 数据值 -> 存在空间中的数值

为了更好地理解数据类型 变量名 = 数据值;这样的变量定义格式,我们需要进一步学习:数据类型 -> 标识符(与变量名有关) -> 键盘录入(与数据值有关)

5’数据类型

分类:Java中的数据类型分为,本质区别是占用内存不同,直接区别是数据取值范围不同

  • 基本数据类型(4类8种)
    • 整数类型
      • byte -> 1Byte,[-128,127]
      • short -> 2Bytes,[-32768,32767]
      • int -> 4,[-2^31^,2^31^-1]
      • long -> 8,[-2^63^,2^63^-1],要使用long类型,就要加’L’后缀
    • 浮点数类型(小数类型)
      • float -> 4,要使用float类型,就要加’F’后缀
      • double -> 8
    • 字符类型
      • char -> 2
    • 布尔类型
      • boolean -> 1Byte,只有true或false
  • 引用数据类型

取值范围大小关系

  • double>float>long>int>short>byte

6’标识符

给类,方法,变量起的名字

硬性要求

  • 由数字,字母,下划线_和美元符$组成
  • 不能以数字开头
  • 不能是关键字
    • 注意,大写了的关键字单词就不是关键字了,比如Class
  • 区分大小写

软性建议

  • 小驼峰命名法 -> 针对方法,变量
    • 标识符是一个单词的时候,全部小写.比如name
    • 标识符由多个单词组成的时候,第一个单词首字母小写,后面单词首字母大写.比如studentName
  • 大驼峰命名法 -> 针对类
    • 标识符是一个单词的时候,首字母大写.比如Student
    • 标识符由多个单词组成的时候,每个单词首字母大写.比如GoodStudent

记得要见名知意!

第三章-运算符

1’算术运算符

包含五种运算

  • 加+ : 如1+1=2
  • 减- : 如3-2=1
  • 乘* : 如2*3=6
  • 除/ : 如10/2=5
    • 若全是整数运算,除不尽,截取余数,保留整数位,如10/3=3
    • 若有浮点数参与运算,会保留余数,如10.3/3=3.333333333335
  • 取模%(取余) : 如10%2=0;2%5=2;
    • 浮点数不能取模
    • 应用场景:
      • 判断某个数是否为偶数
      • 判断两个数间能否整除
      • 对某个范围的循环遍历(如0~3,i%3就可以循环遍历0-3)

特殊情况

  • 如果有浮点数参与计算,结果有可能不精确

除与模一起使用,就可以取得大整数的每一位

1.1’类型转换

隐式转换(自动类型提升)

  • 把取值范围小的数据,转化成取值范围大的数据
    • 如int a = 10; double b = a;把a赋给b时自动提升了值的类型
  • 类型提升规则:
    • 取值范围小的,和取值范围大的进行运算,小的会先提升为大的,再进行运算
    • byte short char 三种类型的数据在运算到时候,会先直接提升为int,然后再继续运算

强制转换

  • 强制性把取值范围大的数据,赋值给取值范围小的变量
    • 如double b = 10.0; int a = b;就会报错,必须强转
  • 格式:目标数据类型 变量名 = (目标数据类型)被强转的数据
    • 如int num = (int)13.14;
  • 结果可能出现精度错误

1.2’算术运算符的加法妙用

数字相加

  • 数字进行运算时,数据类型不一样不能计算,必须先转化成一样的才能进行运算(包括加减乘除)

字符串相加

  • 当+操作中出现字符串时,这个+就会变成字符串连接符而不是算术运算符,会将前后两边的数据进行拼接,产生一个新的字符串
    • 如”123"+123 -> ”123123”
  • 如果有连续的+操作,则从左到右逐个执行
    • 如1+99+"长寿" -> “100长寿”

字符相加

  • 当字符+字符/数字时,会先把字符通过ASCII码表转化成对应的整数再继续运算
    • ASCII码表 -> 65-90对应’A’-‘Z’,97-122对应’a’-‘z’

2’自增自减运算符

分类

  • ++ -> 变量的值+1
  • -- -> 变量的值 -1

使用结果

  • 如果是单独使用,无论放前面还是后面,结果都是+1或-1
  • 如果参与运算,a++/a--表示先用a参与运算再对a+1/-1,即先用后加;反之先a+1/-1再用a参与运算,即先加后用

3’赋值运算符

分类

  • = ,赋值 ,如int a = 10;把10这个数据赋给a变量

以及一些扩展的赋值运算符

  • += ,由于原理相同,就以+=为例说明扩展赋值运算符,a += 10 <=> a = a + 10;
  • -=
  • *=
  • /=
  • %=

使用细节

  • 扩展赋值运算符+=,-=,*=,/= 在底层隐藏了一个强制类型转换
    • short s = 10; s += 10; 也就是s = s + 10;右边short参与运算数值转为int,可是int又赋给s(short类型)没有报错?因为隐藏了强制类型转换

4’关系运算符

也称 ”比较运算符”

分类 (左边a,右边b为例)

  • == ,判断a和b的值是否相等,相等为true,不等为false
  • != ,判断a和b的值是否不相等,不相等为true,相等为false
  • > xxx
  • >= xxx
  • < xxx
  • <= xxx

注意事项

  • 关系运算符/比较运算符的结果都是boolean类型,要么true要么false
  • 不要把== 误写成=

5’逻辑运算符

分类

  • &,逻辑与 -> 并且,两边都为真,结果才为真
  • |,逻辑或 -> 或者,两边都为假,结果才为假
  • ^,逻辑异或 -> 两边相同为false,两边不同为true
  • !,逻辑非 -> 取反

无论左边结果如何,右边表达式都要执行,没有短路效果

5.1’短路逻辑运算符

存在意义:短路实现代码逻辑的判断,提高效率

分类

  • &&,短路与,结果和&相同,但有短路效果
    • 比如a||b,a为真时不必判断b是否为真,直接确认结果为true
  • ||,短路或,结果和|相同,但有短路效果
    • 比如a&&b,a为假时不必判断b是否为假,直接确认结果为false

短路效果

  • 当左边的表达式能确认结果,就不必判断右边的表达式

6’三元运算符

又称三元表达式

作用

  • 进行条件的判断,根据判断的结果得到不同的内容

格式

  • 表达式 ? 表达式1 : 表达式2;

  • 根据关系表达式的结果选择内容,将其赋给变量

    • 如果表达式为true则选择表达式1
    • 如果表达式为false则选择表达式2

  • 例如int max = a > b ? a : b;

    • 如果a>b,max=a
    • 如果b>a,max=b

7’运算符的优先级

背是很难的,但是现在可以记住一点

  • 括号() 的优先级最高,可以利用()来选择自己想要提升运算优先级的部分

优先级排序

  1. . () {}
  2. ! - ++ --
  3. * / %
  4. + -
  5. << >> >>>
  6. < <= > >= instanceof
  7. == !=
  8. &
  9. ^
  10. |
  11. &&
  12. ||
  13. ?:
  14. = += -= *= /= %= &=

2.1’if判断语句

作用:判断表达式结果,然后选择执行不同的代码块

完整版格式

执行流程

  1. 判断第一个表达式结果,为true则执行代码块1然后退出本if语句;为false则继续向下执行
  2. 判断第二个表达式结果,为true则执行代码块2然后退出本if语句;为false则继续向下执行
  3. 判断第n个表达式结果,为true则执行代码块n然后退出本if语句;为false则继续向下执行

    • if(flag == true) ×

2.2’switch选择语句

作用:根据表达式结果,然后选择执行对应代码块

格式

JDK12之后,就有加强版switch语句;格式如下

关键字解释

  • switch 代表这部分switch选择语句
  • case 代表某个值部分的代码块
  • break 表示结束switch语句
  • default 表示所有值都不匹配时,执行的部分代码块

执行流程

  3. 如果所有值都判断完不匹配,就执行default部分的语句体n+1,执行完退出switch语句

注意事项

  • default
    • 位置:位置可以放在switch语句中任意地方,但是建议放最后
    • 省略:可以不写default语句块,语法没问题,但是不建议省略(没备胎了)
  • case穿透
    • 产生原因:语句体中不写break语句
    • 导致结果:当匹配到某个语句体时,会从该语句体开始向下执行,除非遇到break语句或者遇到大括号}或者执行完毕,否则就会执行完下面的所有代码块
    • 使用场景:当多个case的语句体重复(比如1-3月春天xx),就可以用case穿透
  • switch新特性(JDK12->)在上面有
  • switch选择语句与if判断语句的使用场景选择
    • if判断语句一般用于对范围的判断
    • switch选择语句一般用于对有限数据的列举选择

格式

3.2’while循环语句

格式

第五章-数组

1’介绍

数组就是一种容器,可以存放同种数据类型的多个数据

  • 这里的”同种数据类型”可以包含隐式类型转换

建议:容器的类型与存储数据的类型保持一致

2’定义与静态初始化

数组的定义

  • 数据类型[] arr -> 用的多
  • 数据类型 arr[] -> 了解即可

静态初始化

  • 初始化:在内存中,为数组开辟空间,并将数据存入数组的过程

  • 完整格式

    数据类型[] 数组名 = new 数据类型[]{元素1,元素2,元素3,...};

    例如:int[] arr = new int[]{1,2,3};

  • 简化格式

    数据类型[] 数组名 = {元素1,元素2,元素3,...};

    例如:int[] arr = {1,2,3};

  • 注意点

    • 数组一旦初始化,长度就固定了,不能再改变长度

3’数组的地址值

数组名代表数组容器在内存中的地址值

地址值的格式

[D@776ec8df

  • [表示当前是一个数组
  • D表示当前数组类型为double
    • I表示int
  • @表示一个间隔符号(固定格式)
  • 776ec8df数组真正的地址值(十六进制)

4’数组元素访问

格式

  • 数组名[索引];
    • 索引即数组的下标/角标,从0开始,逐个加1

可以利用索引,继续数组元素的访问与赋值

5’数组遍历

数组遍历

  • 将数组中所有的内容取出来,可以进行打印,求和,判断,赋值…
  • 数组遍历要记得↓
    • 遍历的最大索引是arr.length-1
    • 输入arr.fori自动生成有关数组arr的for循环

遍历

  • 指的是取出数据的过程,不要狭义地理解为遍历是打印

6’动态初始化

作用

  • 初始化时只指定数组的长度,由系统为数组分配默认初始值
    • 这样就不用先为每个元素赋值了
    • 默认初始值
      • 整数类型 -> 0
      • 小数类型 -> 0.0
      • 字符类型 -> ‘/u0000’ 空字符
      • 布尔类型 -> false
      • 引用数据类型 -> null

格式

  • 数据类型[] 数组名 = new 数据类型[数组长度];

    比如int[] arr = new int[10];

动态初始化和静态初始化的区别

  • 动态初始化 -> 手动指定数组长度,由系统给出默认初始化值
    • 只知道元素个数,不确认具体数值时使用动态初始化
  • 静态初始化 -> 手动指定数组元素,由系统计算给出数组长度
    • 明确了数据元素数据,直接使用静态初始化

第六章-方法

1’介绍

方法(method)是程序中最小的执行单元,又称函数

作用

  • 提高代码复用性
    • 打包重复代码或特定功能代码,直接调用即可
  • 提高代码可维护性

2’方法的格式

方法就是把一些代码==打包==起来,需要的时候就==调用==

2.1’定义

将一些代码打包到一起的过程,称为方法的定义

格式

  • 如果没有返回值,返回值类型就写void ;return语句就写return;或者不写
  • 方法与方法之间是平级关系,不可嵌套定义

2.2’调用

手动调用使用某个方法的过程,称为方法的调用

格式

方法名(实参列表)

  • 如果有返回值,还要定义一个变量接收返回值
  • 调用时,实际参数的数量和类型必须与方法定义中的形参列表一一对应!

3’方法的重载

作用

  • 同一个方法名实现同种类型的功能

是什么

  • 同一个类中,定义了多个具有同种功能的同名方法,每个方法具有不同的参数类型和参数个数,这些同名的方法就构成重载关系
  • 简单地说:方法的重载就是同一个类中,方法名相同,参数不同,与返回值无关
    • 参数不同包括:个数不同,类型不同,顺序不同
