1’HTML

即HyperText Market Language , 超文本标记语言

  • 超文本
    • 即超越文本 , 其内容不止简单的文本内容,还有多媒体内容,超链接,图片等等
  • 标记语言
    • 即标签语言 , 利用标签实现内容的展示 ,用<>包裹标签

标签的语法

  • 标签成对出现,中间包裹内容
    • 单标签就只有单个,比如<br>,<hr>
    • 成对出现的标签叫做双标签
  • <>里面放英文字母(标签名)
  • 结束标签比开始标签多一个/

标签的关系

  • 父子/嵌套关系 -> html与body ul与li
  • 兄弟/并列关系 -> head与body

HTML网页

  • 本质上就是一个.html文件,经过浏览器渲染解析成美观的网页而已

HTML文件中的注释写法

  • <!-- 注释内容 --> (用快捷键ctrl+/亦可)

2’HTML基本骨架

HTML基本骨架就是网页的模板

code

标签解释

  • html -> 整个网页
  • head -> 网页头部,用来存放给浏览器看的代码
  • body -> 网页主体,用来存放给用户看的代码
  • title -> 网页标题

快捷键

  • 在VS Code中输入! + Tab/Enter 快速生成基本骨架

3’标题标签

用法

  • 新闻/文章/网页区域的大标题

格式

  • <h1>~</h6>

分类

  • 共1~6,六级标题(如果超过6,与默认文本效果一致)

特点

  • 双标签,有开始与结束标签
  • 独占一行(会自动换行)
  • 默认加粗效果
  • 字号从1级到6级逐级递减

4’段落标签

格式

  • <p>段落内容</p>

特点

  • 段落结束自动换行
  • 段落与段落之间存在间隙

5’常用单标签

换行标签

  • <br>

水平线标签

  • <hr>

6’文本格式化标签

作用

  • 为文本添加特殊格式,以突出重点

分类

  • 加粗
    • <strong>
    • <b>
  • 倾斜
    • <em>
    • <i>
  • 下划线
    • <ins>
    • <u>
  • 删除线
    • <del>
    • <s>

7’图像标签

作用

  • 在HTML文件中插入图片

格式

  • <img src = “url" >

常用属性

  • src -> 指定图像的位置和名称,是必须属性
  • alt -> 替换文本,当图片不能显示时展示的文字内容
  • title -> 提示文本,鼠标悬停于图片时显示的内容
  • width/height -> 图片的宽度/高度(一般不在标签中写,而是CSS文件)

8’路径

路径

  • 查找文件时,从起点->终点的路线

根据起点的不同,路径分为

  • 相对路径(☆)
  • 绝对路径

相对路径

  • 从当前位置出发开始找目标

写法

  • /category 表示进入某个文件夹(也就是下一级文件夹)
  • ./category 表示进入当前文件所在文件夹
  • ../category 表示进入当前文件的上一级文件夹

绝对路径

  • 本地查找 -> 从盘符(Windows)/根目录(Mac)位置出发开始找目标

    (Windows默认,其他系统默认/,建议用/)

  • 网络查找 -> 从网站域名出发开始找目标

写法

  • 本地查找

    C:/category/target.html

  • 网络查找

    https://pandac.top/img/avatar.jpg

9’超链接标签

作用

  • 在HTML文件中插入外部链接,点击可以跳转访问

    不仅可以跳转网站,还可以跳转到本地文件

格式

  • <a herf="url”>显示内容</a>

常用属性

  • href -> 指定跳转地址,是必须属性
    • 开发初期,可以写url为#,表示空链接
  • target -> 跳转方式,属性值↓↓
    • _self -> 原窗口打开(默认值)
    • _blank -> 新窗口打开

10’音频标签

作用

  • 在HTML文件中插入音频

格式

  • <audio src = “url" ></audio>

常用属性

  • src -> 指定音频资源地址,是必须属性
  • controls -> 显示音频控制面板
    • HTML5 -> 若属性名==属性值,可以简写成一个~
  • loop -> 循环播放
  • autoplay -> 自动播放(一般默认被浏览器禁用)

11’视频标签

作用

  • 在HTML文件中插入视频

格式

  • <video src = “url" ></video>

常用属性

  • src -> 指定视频资源地址,是必须属性
  • controls -> 显示视频控制面板
  • muted -> 静音播放
  • loop -> 循环播放
  • autoplay -> 自动播放(仅支持在静音状态下自动播放)

12’列表

作用

  • 布局内容排列整齐的区域

分类

  • 无序列表(★)
  • 有序列表
  • 定义列表

无序列表unordered list

作用

  • 布局排列整齐,不需要规定顺序的内容区域

格式

ul嵌套li,ul代表无序列表,li代表列表条目

注意事项

  • ul标签里只能写li标签
  • li标签中可以包含任何内容

有序列表ordered list

作用

  • 布局排列整齐,需要规定顺序的内容区域

格式

ol嵌套li,ol代表有序列表,li代表列表条目

注意事项

  • ol标签里只能写li标签
  • li标签中可以包含任何内容

定义列表

作用

  • 一般布局于网页底部(像帮助中心),一个标题对应多个内容

格式

dl嵌套dt和dl,dl是定义列表,dt是列表标题,dl是列表的内容

注意事项

  • dl标签里只能写dt和dd标签
  • dt和dd标签中可以包含任何内容

13’表格

基本使用

格式

table嵌套tr,tr嵌套th和td,table是表格,tr是表格中的行,th代表表头单元格,td代表内容单元格

效果

标题1 标题2
单元格1-1 单元格1-2
单元格2-1 单元格2-2

注意事项

  • 表格默认无边框线,需要在table标签里加上border属性,为表格添加边框线

表格结构标签(了解)

作用

  • 为表格的内容划分区域,让表格结构更清晰

分类

  • thead -> 表格头部
  • tbody -> 表格主体
  • tfoot -> 表格底部

合并单元格

作用

  • 将多个单元格合并成一个单元格

用法

  • 跨行合并 -> 合并同列方向单元格
  • 跨列合并 -> 合并同行方向单元格

步骤

  • 跨行合并,保留最上单元格,并在该td标签中添加属性rowspan ,属性值为要合并单元格的个数
  • 跨列合并,保留最左单元格,并在该td标签中添加属性colspan ,属性值为要合并单元格的个数

注意事项

  • 不能跨结构标签合并内容

14’表单

作用

  • 收集用户传入信息

使用场景

  • 登录/注册界面
  • 搜索区域

input标签

input标签利用type属性值的不同,而形成不同的功能

格式

  • <input type="...">

常用type属性值(功能)

  • text -> 文本框
    • 用于输入单行文本
  • password -> 密码框
    • 用于输入密码
    • 会自动隐藏输入字符
  • radio -> 单选框
    • 多个单选框用同一个name属性实现多个只能选一个
    • checked属性默认选中某一个选项
  • checkbox -> 多选框(复选框)
    • checked默认选中
  • file -> 上传文件
    • 添加multiple属性实现文件多选上传

常用属性

  • placeholder -> 占位文本
    • 作用 : 添加提示信息(给文本框和密码框使用)
    • 在框的底部显示浅灰色提示文本

下拉菜单select

格式

select嵌套option,select是下拉菜单整体,option是每一个选项

常用属性

  • selected默认选择

文本域textarea

作用

  • 输入多行文本

格式

  • <textarea>提示文字</textarea>
  • 文字可以换行
  • 大小可以拖拽变换(要禁用它)

label标签

作用

  • 给标签添加说明文字
  • 当label标签与表单控件绑定时,可以增大表单控件的点击范围

格式

  • 一个label标签绑定一个表单控件
  • 完整写法&简化写法↓↓

按钮button

格式

  • <button type="">按钮</button>

常用type属性值

  • submit -> 提交按钮(默认功能)
  • reset -> 重置按钮,重置表单控件为默认值
    • 要配合form表单区域标签使用,才能重置输入数据
  • button -> 普通按钮,配合JS使用

15’无语义布局标签

作用

  • 布局网页(划分区域,摆放内容),但是没有实际语义

分类

  • div标签(大盒子)
    • 独占一行(自动换行)
  • span标签(小盒子)
    • 一行多个(不换行)

16’字符实体

作用

  • 在网页中显示预留字符(有定义具体功能的字符)

常用字符实体

  • &nbsp; -> 空格
  • &lt; -> 小于号 <
  • &gt; -> 大于号 >
  • &quot; -> 引号 “
  • &amp; -> &符号 &
  • &trade; -> 商标 ™
  • &copy; -> 版权 ©
  • &reg; -> 已注册 ®
