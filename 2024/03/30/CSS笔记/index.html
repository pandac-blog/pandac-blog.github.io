CSS笔记

发表于2024-03-30|更新于2024-04-01|前端CSS
|字数总计:1.3k|阅读时长:5分钟|阅读量:

1’CSS

即Cascading Style Sheets,层叠样式表

用来美化HTML文档

CSS代码书写

  • 选择器{css代码}

  • css代码其实就是n对键值对 (属性名:属性值;)

2’样式引入方式

内部样式表(内联样式表)

  • 将CSS代码放在title标签下的style标签中
  • 学习时使用

外部样式表

  • 将CSS代码放在.css文件中,再用link标签引入HTML文档

行内样式表

  • 将CSS代码写在标签的style属性中

3’基础选择器

作用

  • 查找标签,设置样式

标签选择器

格式

  • 标签名{css代码}

作用

  • 选中HTML文档中同名的所有标签,设置相同样式

类选择器

格式

  • css代码 -> .类名{css代码}
  • 选中标签 -> 添加class=”类名”属性

作用

  • 查找标签,差异化设置标签样式

  • 一个类选择器可以给多个标签用

  • 一个标签可以加多个class属性值,用空格隔开

    <div class="test size">div</div

id选择器

格式

  • css代码 -> #id{css代码}
  • 选中标签 -> 添加id=”id值”属性

作用

  • 查找标签,差异化设置标签样式
  • 一个id选择器在一个页面只能使用一次
  • 一般配合JS使用,很少用于CSS设置样式

通配符选择器

格式

  • *{css代码}

作用

  • 选中HTML文档中所有标签,设置相同样式
  • 一般用于初始化所有标签的默认样式(清除)

4’字体修饰属性

字体大小font-size

格式

  • 属性名font-size
  • 属性值:文字尺寸
    • 常用单位 -> 像素px

例子

<div style="font-size:20px">div</div>

字体粗细font-weight

格式

  • 属性名font-weight
  • 属性值
    • 数字 -> 400正常,700加粗
    • 关键字 -> normal正常,bold加粗

例子

<div style="font-weight:700">div</div>

字体倾斜font-style

格式

  • 属性名font-style
  • 属性值
    • 正常normal -> 不倾斜
    • 倾斜italic -> 倾斜
      • 一般不使用倾斜效果

例子

<div style="font-style:italic">div</div>

行高line-height

作用

  • 设置多行文本之间的间距

格式

  • 属性名line-height
  • 属性值
    • 数字+px -> 写多大就多大
    • 数字 -> 当前标签font-size属性的倍数

例子

<div style="line-height:2">div</div>

组成内容

  • 上间距(第一个红色框部分)
  • 文本高度(绿色框部分)
  • 下间距(第二个红色框部分)

根据等价效果,第一行的下间距到第二行的下间距其实也是行高的值 -> 行高的测量方法

垂直居中

  • 设置行高属性值=盒子高度属性值 => 使文本在标签中居中
    • 仅适用于单行文字

字体族font-family

格式

  • 属性名font-family
  • 属性值: 字体名

例子

<div style="font-family:楷体">div</div>

多字体设置

font-family属性可以设置多个字体名,用”,”隔开,从左向右执行,不能用就换下一个,一般最后设置sans-serif字体(无衬线字体)

字体复合属性font(了解)

格式

  • 属性名font
  • 属性值:是否倾斜 是否加粗 字号/行高 字体 (按顺序写)
    • 字号和字体必须写,否则font不生效

例子

<div style="font:normal 400 20px/10px 宋体">div</div>

5’文本修饰属性

文本缩进text-indent

格式

  • 属性名text-indent
  • 属性值
    • 数字+px
    • 数字+em (1em <=> 当前标签字号大小)

例子

<div style="text-indent:2em">div</div> -> 首行缩进2个字~

文本对齐text-align

格式

  • 属性名text-align
  • 属性值
    • left(默认左对齐)
    • center
    • right

例子

<div style="text-align:center">div</div>

作用

  • 对内容居中,而非标签元素居中

修饰线text-decoration

格式

  • 属性名text-decoration
  • 属性值
    • 无(可去除超链接默认下划线) -> none
    • 下划线 -> underline
    • 删除线 -> line-through
    • 上划线(基本不用) -> overline

例子

<div style="text-decoration:underline">div</div>

颜色color

格式

  • 属性名color
  • 属性值(默认对应红-绿-蓝三原色取值,从0~255)
    • 关键字 -> blue,red,green
    • 十六进制表示法 -> #aabbcc 可简写为#abc (开发常使用十六进制表示)
    • rgb表示法 -> rgb(r,g,b), 即红,绿,蓝的取值
      • r,g,b取值范围0~255
    • rgba表示法 ->rgba(r,g,b,a), 即红,绿,蓝,透明度的取值 (主要用来描述透明色)
      • a取值0~1

6’进阶选择器

复合选择器

介绍

  • 由多个基础选择器,通过不同方式组合而成的选择器

作用

  • 更精确地选择标签,并设置样式
