1’CSS

即Cascading Style Sheets,层叠样式表

用来美化HTML文档

CSS代码书写

  • 选择器{css代码}

  • css代码其实就是n对键值对 (属性名:属性值;)

2’样式引入方式

内部样式表(内联样式表)

  • 将CSS代码放在title标签下的style标签中
  • 学习时使用

外部样式表

  • 将CSS代码放在.css文件中,再用link标签引入HTML文档

行内样式表

  • 将CSS代码写在标签的style属性中

3’基础选择器

作用

  • 查找标签,设置样式

标签选择器

格式

  • 标签名{css代码}

作用

  • 选中HTML文档中同名的所有标签,设置相同样式

类选择器

格式

  • css代码 -> .类名{css代码}
  • 选中标签 -> 添加class=”类名”属性

作用

  • 查找标签,差异化设置标签样式

  • 一个类选择器可以给多个标签用

  • 一个标签可以加多个class属性值,用空格隔开

    <div class="test size">div</div

id选择器

格式

  • css代码 -> #id{css代码}
  • 选中标签 -> 添加id=”id值”属性

作用

  • 查找标签,差异化设置标签样式
  • 一个id选择器在一个页面只能使用一次
  • 一般配合JS使用,很少用于CSS设置样式

通配符选择器

格式

  • *{css代码}

作用

  • 选中HTML文档中所有标签,设置相同样式
  • 一般用于初始化所有标签的默认样式(清除)

4’字体修饰属性

字体大小font-size

格式

  • 属性名font-size
  • 属性值:文字尺寸
    • 常用单位 -> 像素px

例子

<div style="font-size:20px">div</div>

字体粗细font-weight

格式

  • 属性名font-weight
  • 属性值
    • 数字 -> 400正常,700加粗
    • 关键字 -> normal正常,bold加粗

例子

<div style="font-weight:700">div</div>

字体倾斜font-style

格式

  • 属性名font-style
  • 属性值
    • 正常normal -> 不倾斜
    • 倾斜italic -> 倾斜
      • 一般不使用倾斜效果

例子

<div style="font-style:italic">div</div>

行高line-height

作用

  • 设置多行文本之间的间距

格式

  • 属性名line-height
  • 属性值
    • 数字+px -> 写多大就多大
    • 数字 -> 当前标签font-size属性的倍数

例子

<div style="line-height:2">div</div>

组成内容

  • 上间距(第一个红色框部分)
  • 文本高度(绿色框部分)
  • 下间距(第二个红色框部分)

根据等价效果,第一行的下间距到第二行的下间距其实也是行高的值 -> 行高的测量方法

垂直居中

  • 设置行高属性值=盒子高度属性值 => 使文本在标签中居中
    • 仅适用于单行文字

字体族font-family

格式

  • 属性名font-family
  • 属性值: 字体名

例子

<div style="font-family:楷体">div</div>

多字体设置

font-family属性可以设置多个字体名,用”,”隔开,从左向右执行,不能用就换下一个,一般最后设置sans-serif字体(无衬线字体)

字体复合属性font(了解)

格式

  • 属性名font
  • 属性值:是否倾斜 是否加粗 字号/行高 字体 (按顺序写)
    • 字号和字体必须写,否则font不生效

例子

<div style="font:normal 400 20px/10px 宋体">div</div>

5’文本修饰属性

文本缩进text-indent

格式

  • 属性名text-indent
  • 属性值
    • 数字+px
    • 数字+em (1em <=> 当前标签字号大小)

例子

<div style="text-indent:2em">div</div> -> 首行缩进2个字~

文本对齐text-align

格式

  • 属性名text-align
  • 属性值
    • left(默认左对齐)
    • center
    • right

例子

<div style="text-align:center">div</div>

作用

  • 对内容居中,而非标签元素居中

修饰线text-decoration

格式

  • 属性名text-decoration
  • 属性值
    • 无(可去除超链接默认下划线) -> none
    • 下划线 -> underline
    • 删除线 -> line-through
    • 上划线(基本不用) -> overline

例子

<div style="text-decoration:underline">div</div>

颜色color

格式

  • 属性名color
  • 属性值(默认对应红-绿-蓝三原色取值,从0~255)
    • 关键字 -> blue,red,green
    • 十六进制表示法 -> #aabbcc 可简写为#abc (开发常使用十六进制表示)
    • rgb表示法 -> rgb(r,g,b), 即红,绿,蓝的取值
      • r,g,b取值范围0~255
    • rgba表示法 ->rgba(r,g,b,a), 即红,绿,蓝,透明度的取值 (主要用来描述透明色)
      • a取值0~1

6’进阶选择器

复合选择器

介绍

  • 由多个基础选择器,通过不同方式组合而成的选择器

作用

  • 更精确地选择标签,并设置样式

后代选择器

选中某个元素的(所有)后代元素

格式

  • 父选择器 子选择器{css代码}

子代选择器

选中某个元素的子代元素(只选中儿子)

格式

  • 父选择器 > 子选择器{css代码}

并集选择器

选中多组标签设置相同样式

格式

  • 选择器1,选择器2,...,选择器{css代码}

交集选择器

选中同时满足多个条件的元素

格式

  • 选择器1选择器2{css代码}

    直接连着写,不用输入任何字符

    如果有标签选择器,就要将标签选择器写在最前面

伪类选择器

伪类表示元素的状态

伪类选择器就是对元素的某个状态设置样式

鼠标悬停状态

鼠标悬停状态

  • 选择器:hover{css代码}

    当鼠标悬停在此选择器对应的标签上时,该标签元素就会有状态样式显示

任何标签都可以设置鼠标悬停状态

超链接伪类选择器拓展

超链接有四个常用伪类选择器

  • 选择器:link{css代码}

    访问前的状态

  • 选择器:visited{css代码}

    访问后的状态

  • 选择器:hover{css代码}

    鼠标悬停的状态

  • 选择器:active{css代码}

    点击时的状态

如果都要写,一定按照 l->v->h->a 的顺序书写伪类选择器

结构伪类选择器

作用

  • 根据元素的结构关系查找元素

E:first-child

查找第一个E元素

E:last-child

查找最后一个E元素

E:nth-child(N)

查找第N个E元素(第一个元素N认为1)

公式写法

则可根据元素的结构关系查找多个元素

  • 偶数 -> 2n
  • 奇数 -> 2n+1 / 2n-1
  • 找到倍数为5的标签 -> 5n
  • 找到第5个以后的标签 -> n+5
  • 找到第5个以前的标签 -> -n+5

伪元素选择器

作用

  • 通过CSS创建虚拟元素(伪元素),用来摆放装饰性内容

常用

  • E:before -> 在E元素中,最前面添加一个伪元素
  • E:after -> 在E元素中,最后面添加一个伪元素

注意事项

  • 必须设置content:””属性,来设置伪元素的内容
  • 伪元素默认行内显示模式
  • 权重与标签选择器同等

7’CSS三大特性

继承性

子集标签默认继承父级标签的字体/文本修饰属性

如下例子,为<body>标签设置字体/文本属性,其子标签会默认继承

层叠性

相同的属性会覆盖 -> 后面的CSS属性会覆盖前面的CSS属性

不同的属性会叠加 -> 不同的CSS属性都生效

优先级

也叫做权重

当一个标签使用了多个选择器时,基于不同种类选择器的匹配规则

  • 规则

    谁的优先级高,使用谁的样式

  • 公式

    !important > 行内样式 > id选择器 > 类选择器 > 标签选择器 > 通配符选择器

    (选中范围越大,优先级越低)

复合选择器优先级的叠加计算规则

暂不需要,要用时再补

8’Emmet写法

一种用于快速编写 HTML 和 CSS 代码的工具，它使用简洁的缩写语法来生成代码

HTML缩写

  • 标签名.类名 -> 生成一个标签且标签中自动写入类名
  • 标签名#id值 -> 生成一个标签且标签中自动写入id值
  • div+p -> 生成一个div标签和一个p标签
  • div>p -> 生成一个div标签,并嵌套一个p标签
  • span*3 -> 生成三个span标签
  • div{内容} -> 生成一个div标签,且自动写入内容

CSS缩写

大多数简写方式是属性单词的首字母

比如bgc -> background-color

h50+w50 -> height:50px;width:50px;

9’背景属性

背景色background-color

格式

  • 属性名background-color
  • 属性值 -> color属性值

背景图background-image

格式

  • 属性名background-image
  • 属性值 -> url(图像的URL);

作用

  • 在背景处实现装饰性图片

背景图平铺方式background-repeat

格式

  • 属性名background-repeat

  • 属性值

    no-repeat不平铺

    repeat平铺(其实就是复制,也是默认效果)

    repeat-x水平方向平铺

    repeat-y垂直方向平铺

作用

  • 控制背景图片的展示效果

背景图位置background-position

格式

  • 属性名background-position

  • 属性值

    关键字

    • left,right,center
    • top,bottom

    坐标 -> 数字+px

    ​ 坐标以水平向右为正x轴,水平向下为正y轴

注意事项

  • 如果写两个参数

    第一个参数代表水平方向,第二个参数代表垂直方向

  • 关键字取值可以颠倒第一个和第二个参数的位置

    如bottom left

  • 如果只写一个参数

    只写一个关键字,另一个方向默认居中

    只写一个数字表示水平方向,垂直方向默认居中

背景图缩放background-size

格式

  • 属性名background-size

  • 属性值

    关键字

    • cover -> 等比例缩放背景图片以完全覆盖背景区,保证了背景区的全部填充,有可能部分背景图看不见
    • contain -> 等比例缩放背景图片以完全装入背景区,保证了背景图的全部展示,有可能部分背景区空白

    百分比 -> 根据盒子尺寸计算图片大小,保证宽度和盒子一致,高度等比例缩放

    数字+单位

背景图固定background-attachment

格式

  • 属性名background-attachment
  • 属性值 -> fixed

作用

  • 使背景图片不随着元素的内容滚动

    就比如我的网站首页,炫彩色背景图就不会随着元素内容的移动而移动

背景复合属性background

格式

  • 属性名background
  • 属性值背景色 背景图 背景图平铺方式 背景图位置/背景图缩放 背景图固定 (空格隔开各属性值,不区分先后顺序)

10’元素的显示模型

即标签/元素的显式方式

作用

  • 布局网页时,根据标签的显示模式选择合适的标签布局网页

分类

  • 块级元素
  • 行内元素
  • 行内块元素

块级元素block

独占一行

宽度默认父级100%

加宽高属性生效

行内元素inline

一行多个

尺寸由内容撑开

加宽高属性不生效(无效)

行内块元素inline-block

一行多个

默认尺寸由内容撑开

加宽高属性生效

转换显示模式

属性名display

属性值

  • block 块级
  • inline-block 行内块
  • inline 行内

11’盒子模型

作用

  • 布局网页,摆放盒子和内容

组成部分

  • 内容区域 - width&height
  • 内边距 - padding
  • 边框线 - border
  • 外边距 - margin
